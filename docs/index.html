Campus Navigation Project

The campus navigation app is a powerful tool designed to enhance the user experience for students, faculty, and visitors by offering real-time, location-based guidance across university grounds. Through AI-powered functions and seamless navigation, the app allows users to quickly locate essential facilities such as restrooms, study spaces, and dining halls. Additionally, it provides real-time updates and notifications, including alerts about campus events, building closures, and construction activities. This helps users stay informed and reduces the time spent searching for key locations. Beyond basic navigation, the app also serves as an information hub with an AI assistant that provides responses to user queries, making campus resources more accessible. Users can ask questions such as “Where is the nearest dining hall?” or “What events are happening today?” The AI integration not only promotes convenience and engagement but also fosters productivity by delivering instant access to important information. This intuitive combination of navigation and knowledge-sharing enhances campus life, offering an improved sense of community and efficiency.

Installation

  1. Step 1: Install Node.js
    • Using the Node.js Installer: Download Link
  2. Step 2: Install the App
    • Clone the Repository: git clone https://github.com/RamonZubiate/TTU-FourLoop
    • Install Dependencies: cd TTU-FourLoop/app && npm install
  3. Step 3: Platform-Specific Setup for Environment
    • iOS Setup (Mac Required):
      • Install Homebrew: /bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"
      • Run: brew install cocoapods
      • Navigate to the iOS folder and run: pod install
      • Download Xcode from the App Store and open the workspace: open TTU-FourLoop/app/ios/app.xcworkspace
      • Select a simulator in Xcode (e.g., iPhone 14).
    • Android Setup:
      • Download Android Studio and open the Android folder in Android Studio.
      • Install Java JDK 21 or higher.
      • Set up an Android Emulator (Pixel 4 recommended).
      • Set up an Android SDK following the guide: SDK Setup Guide
      • Within Android Studio, you can install the Android 14 SDK as follows:
        • Click Tools > SDK Manager, then click Show Package Details.
        • In the SDK Platforms tab, expand the Android 14.0 ("UpsideDownCake") section and select the Android SDK Platform 34 package.
        • In the SDK Tools tab, expand the Android SDK Build-Tools 34 section and select the latest 34.x.x version.
        • Click Apply > OK to download and install the selected packages.
  4. Step 4: Configure Mapbox Environment (iOS Only)
    • Set your .netrc file in your macOS home directory: Mapbox Installation Guide
    • In a terminal, go to your home directory and check if the .netrc file exists: $ cd ~ $ file .netrc

      If the .netrc file does not exist, run:

      $ touch .netrc $ open .netrc

User Manual

Chat Page

Key Features

  • User Messaging: Send and receive messages through the input field.
  • Chatbot Responses: Automatic responses with distinctive styling.
  • Real-Time Updates: Messages appear instantly in the chat view.
  • Loading Indicator: Spinner displayed while waiting for bot response.
  • Chat History: Displays the last hour of messages.

How to Use

  1. Sending a Message: Type a message and press "Send".
  2. Viewing Bot Responses: The bot responds after a short delay with a different background color.

Events Page

Key Features

  • Dynamic Category Display: Horizontal scrollable cards for categories.
  • Event List: Displays event details like title, image, and date/time.
  • Themed Interface: Consistent theming with accent colors.
  • Real-Time Data Fetching: Dynamically loads events from Firestore.

How to Use

  1. Navigation: Click categories to view places and events.
  2. Learn More: Opens external resources.

Map Screen

Key Features

  • Dynamic Location Tracking: Tracks user location and updates the map.
  • Autocomplete for Destinations: Suggests destinations via Google Places API.
  • Route Calculation: Provides directions via Google Maps API.
  • Interactive Map: Displays routes and markers on Mapbox.
  • Directions Display: Shows turn-by-turn directions once fetched.

How to Use

  1. Destination Input: Enter a destination to get route suggestions.
  2. Travel Mode: Choose walking, cycling, or transit mode.
  3. Start Navigation: Click "Start Navigation" to follow your location.

PlacesList

Key Features

  • Image Gallery Modal: Full-screen photo gallery with navigation controls.
  • Places List: Filtered list of places by category.
  • Star Rating: View the rating for each place.
  • Place Busyness: Shows real-time busyness for certain places.

How to Use

  1. Scrolling: Scroll through places based on categories like Food or Recreation.
  2. Viewing Photos: Click on a photo to open the image gallery.
  3. View Place Busyness: Check the current busyness of a place.
